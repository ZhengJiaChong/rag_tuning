## 概述

### 为什么需要意图识别？

意图识别是理解用户真实需求的关键环节，直接影响后续的处理策略和资源分配。

**核心价值**：
- ✅ **精准路由**：不同意图使用不同的处理管道
- ✅ **资源优化**：简单意图快速响应，复杂意图精细处理
- ✅ **提升体验**：提供符合用户期望的回答方式
- ✅ **降低成本**：避免不必要的计算和检索

### 意图标签设计原则

✅ **MECE原则**（相互独立，完全穷尽）：
- 每个意图应该有清晰的边界
- 所有可能的查询都能被分类

✅ **粒度适中**：
- 太粗：无法差异化处理
- 太细：难以准确分类

✅ **可扩展性**：
- 预留"其他"类别
- 支持动态添加新意图

✅ **业务导向**：
- 意图划分应服务于业务需求
- 不同意图对应不同的处理策略

## 意图识别方法

### 1. 基于规则的方法

```python
import re
from typing import Dict, List

class RuleBasedIntentClassifier:
    """
    基于规则的意图分类器
    
    优势：快速、可解释、易维护
    劣势：覆盖有限、难以处理变体
    """
    
    def __init__(self):
        # 意图模式库
        self.intent_patterns = {
            'password_reset': {
                'patterns': [
                    r'.*密码.*重置.*',
                    r'.*忘记.*密码.*',
                    r'.*找回密码.*',
                    r'.*密码.*忘了.*',
                    r'怎么重置密码',
                ],
                'keywords': ['密码', '重置', '找回', '忘记'],
            },
            'order_status': {
                'patterns': [
                    r'.*订单.*状态.*',
                    r'.*物流.*查询.*',
                    r'.*到哪了.*',
                    r'.*发货.*了吗.*',
                ],
                'keywords': ['订单', '物流', '快递', '发货'],
            },
            'refund_request': {
                'patterns': [
                    r'.*退款.*',
                    r'.*退货.*',
                    r'.*换货.*',
                    r'.*退钱.*',
                ],
                'keywords': ['退款', '退货', '换货', '退还'],
            },
            'product_inquiry': {
                'patterns': [
                    r'.*什么.*功能.*',
                    r'.*怎么用.*',
                    r'.*如何使用.*',
                    r'.*产品介绍.*',
                ],
                'keywords': ['功能', '使用', '介绍', '特点'],
            },
            'technical_support': {
                'patterns': [
                    r'.*错误.*',
                    r'.*报错.*',
                    r'.*失败.*',
                    r'.*bug.*',
                    r'.*异常.*',
                ],
                'keywords': ['错误', '报错', '失败', '异常', 'bug'],
            },
            'greeting': {
                'patterns': [
                    r'^你好$',
                    r'^您好$',
                    r'^hello$',
                    r'^hi$',
                    r'^在吗$',
                ],
                'keywords': ['你好', '您好', 'hello', 'hi'],
            },
        }
    
    def classify(self, query: str) -> Dict:
        """
        分类查询意图
        
        Args:
            query: 用户查询
        
        Returns:
            {
                'intent': 意图类别,
                'confidence': 置信度,
                'method': 分类方法,
                'matched_pattern': 匹配的模式（可选）
            }
        """
        best_intent = None
        best_score = 0
        matched_pattern = None
        
        for intent, config in self.intent_patterns.items():
            score = self._compute_match_score(query, config)
            
            if score > best_score:
                best_score = score
                best_intent = intent
        
        # 如果没有匹配到任何意图，返回unknown
        if best_intent is None or best_score < 0.3:
            return {
                'intent': 'unknown',
                'confidence': 0.0,
                'method': 'rule_based'
            }
        
        return {
            'intent': best_intent,
            'confidence': best_score,
            'method': 'rule_based',
            'matched_pattern': matched_pattern
        }
    
    def _compute_match_score(self, query: str, config: Dict) -> float:
        """计算查询与意图模式的匹配分数"""
        score = 0.0
        
        # 1. 正则匹配
        for pattern in config['patterns']:
            if re.search(pattern, query, re.IGNORECASE):
                score += 0.5
                break
        
        # 2. 关键词匹配
        keywords = config['keywords']
        matched_keywords = [kw for kw in keywords if kw in query]
        
        if matched_keywords:
            keyword_score = len(matched_keywords) / len(keywords)
            score += 0.3 * keyword_score
        
        # 3. 长度惩罚（避免过短查询误判）
        if len(query) < 3:
            score *= 0.5
        
        return min(1.0, score)
    
    def add_intent(self, intent_name: str, patterns: List[str],
                  keywords: List[str]):
        """动态添加新意图"""
        self.intent_patterns[intent_name] = {
            'patterns': patterns,
            'keywords': keywords
        }


# 使用示例
classifier = RuleBasedIntentClassifier()

queries = [
    "忘记密码怎么办",
    "我的订单到哪了",
    "我想退款",
    "你好"
]

for query in queries:
    result = classifier.classify(query)
    print(f"查询: {query}")
    print(f"意图: {result['intent']}")
    print(f"置信度: {result['confidence']:.2f}")
    print()
```

**优点**：
- ✅ 速度快（< 1ms）
- ✅ 可解释性强
- ✅ 易于维护和调试
- ✅ 无需训练数据

**缺点**：
- ❌ 覆盖有限
- ❌ 难以处理表述变体
- ❌ 需要人工维护规则

**适用场景**：
- 意图类别少且固定
- 查询模式规律性强
- 作为第一层快速过滤

### 2. 基于机器学习的方法

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import SVC
from sklearn.pipeline import Pipeline
from sklearn.model_selection import cross_val_score
import joblib
from typing import List

class MLIntentClassifier:
    """
    基于传统机器学习的意图分类器
    
    使用TF-IDF + SVM
    """
    
    def __init__(self, model_path=None):
        self.pipeline = Pipeline([
            ('tfidf', TfidfVectorizer(
                max_features=10000,
                ngram_range=(1, 2),  # 使用unigram和bigram
                min_df=2,
                max_df=0.95
            )),
            ('clf', SVC(
                kernel='linear',
                probability=True,
                C=1.0,
                class_weight='balanced'
            ))
        ])
        
        self.is_trained = False
        self.label_list = []
        
        if model_path:
            self.load(model_path)
    
    def train(self, texts: List[str], labels: List[str]):
        """
        训练分类器
        
        Args:
            texts: 训练文本列表
            labels: 对应的标签列表
        """
        self.label_list = list(set(labels))
        
        # 训练
        self.pipeline.fit(texts, labels)
        self.is_trained = True
        
        # 交叉验证评估
        scores = cross_val_score(self.pipeline, texts, labels, cv=5)
        print(f"训练完成，5折交叉验证准确率: {scores.mean():.4f} (+/- {scores.std() * 2:.4f})")
    
    def predict(self, query: str) -> Dict:
        """
        预测意图
        
        Args:
            query: 用户查询
        
        Returns:
            {
                'intent': 预测的意图,
                'confidence': 置信度,
                'all_probabilities': 所有类别的概率,
                'method': 分类方法
            }
        """
        if not self.is_trained:
            raise Exception("Model not trained yet!")
        
        # 预测
        intent = self.pipeline.predict([query])[0]
        probabilities = self.pipeline.predict_proba([query])[0]
        
        # 获取置信度
        confidence = float(max(probabilities))
        
        # 构建概率字典
        all_probs = dict(zip(self.pipeline.classes_, probabilities))
        
        return {
            'intent': intent,
            'confidence': confidence,
            'all_probabilities': all_probs,
            'method': 'ml_svm'
        }
    
    def save(self, model_path: str):
        """保存模型"""
        joblib.dump({
            'pipeline': self.pipeline,
            'label_list': self.label_list,
            'is_trained': self.is_trained
        }, model_path)
        print(f"模型已保存到: {model_path}")
    
    def load(self, model_path: str):
        """加载模型"""
        data = joblib.load(model_path)
        self.pipeline = data['pipeline']
        self.label_list = data['label_list']
        self.is_trained = data['is_trained']
        print(f"模型已从 {model_path} 加载")


# 使用示例
if __name__ == '__main__':
    # 准备训练数据
    train_texts = [
        "忘记密码怎么办",
        "怎么重置密码",
        "密码忘了",
        "我的订单状态",
        "查询物流信息",
        "订单到哪了",
        "我要退款",
        "申请退货",
        "你好",
        "您好",
        # ... 更多样本（至少每类50-100个）
    ]
    
    train_labels = [
        "password_reset",
        "password_reset",
        "password_reset",
        "order_status",
        "order_status",
        "order_status",
        "refund_request",
        "refund_request",
        "greeting",
        "greeting",
        # ... 对应标签
    ]
    
    # 训练
    classifier = MLIntentClassifier()
    classifier.train(train_texts, train_labels)
    
    # 保存
    classifier.save('intent_classifier.pkl')
    
    # 预测
    query = "我忘记密码了"
    result = classifier.predict(query)
    
    print(f"查询: {query}")
    print(f"意图: {result['intent']}")
    print(f"置信度: {result['confidence']:.4f}")
```

**优点**：
- ✅ 比规则方法泛化能力强
- ✅ 训练速度快
- ✅ 资源占用少
- ✅ 可解释性较好

**缺点**：

**适用场景**：

- 意图类别相对稳定
- 对延迟敏感

### 3. 基于深度学习的方法

```python
import torch
import torch.nn as nn
from transformers import BertTokenizer, BertModel
from torch.utils.data import Dataset, DataLoader
from typing import List

class BertIntentClassifier(nn.Module):
    """
    基于BERT的意图分类模型
    """
    
    def __init__(self, num_labels: int, model_name='bert-base-chinese'):
        super().__init__()
        
        # 加载预训练BERT
        self.bert = BertModel.from_pretrained(model_name)
        
        # Dropout防止过拟合
        self.dropout = nn.Dropout(0.3)
        
        # 分类层
        self.classifier = nn.Linear(768, num_labels)
    
    def forward(self, input_ids, attention_mask):
        """
        前向传播
        
        Args:
            input_ids: token IDs
            attention_mask: 注意力掩码
        
        Returns:
            logits: 未归一化的分数
        """
        outputs = self.bert(
            input_ids=input_ids,
            attention_mask=attention_mask
        )
        
        # 使用[CLS] token的输出
        pooled_output = outputs.pooler_output
        
        # Dropout
        output = self.dropout(pooled_output)
        
        # 分类
        logits = self.classifier(output)
        
        return logits


class IntentDataset(Dataset):
    """意图分类数据集"""
    
    def __init__(self, texts: List[str], labels: List[int],
                 tokenizer, max_length=128):
        self.texts = texts
        self.labels = labels
        self.tokenizer = tokenizer
        self.max_length = max_length
    
    def __len__(self):
        return len(self.texts)
    
    def __getitem__(self, idx):
        text = self.texts[idx]
        label = self.labels[idx]
        
        # Tokenize
        encoding = self.tokenizer.encode_plus(
            text,
            add_special_tokens=True,
            max_length=self.max_length,
            padding='max_length',
            truncation=True,
            return_attention_mask=True,
            return_tensors='pt'
        )
        
        return {
            'input_ids': encoding['input_ids'].squeeze(),
            'attention_mask': encoding['attention_mask'].squeeze(),
            'labels': torch.tensor(label, dtype=torch.long)
        }


class DeepIntentClassifier:
    """
    深度学习意图分类器
    
    基于BERT微调
    """
    
    def __init__(self, model_path=None, label_list: List[str] = None):
        self.label_list = label_list or []
        self.num_labels = len(self.label_list)
        
        if self.num_labels > 0:
            self.model = BertIntentClassifier(self.num_labels)
            self.tokenizer = BertTokenizer.from_pretrained('bert-base-chinese')
            
            if model_path:
                self.load(model_path)
    
    def train(self, train_texts: List[str], train_labels: List[str],
             val_texts: List[str] = None, val_labels: List[str] = None,
             epochs: int = 5, batch_size: int = 32,
             learning_rate: float = 2e-5):
        """
        训练模型
        
        Args:
            train_texts: 训练文本
            train_labels: 训练标签
            val_texts: 验证文本（可选）
            val_labels: 验证标签（可选）
            epochs: 训练轮数
            batch_size: 批次大小
            learning_rate: 学习率
        """
        # 标签编码
        label_to_idx = {label: idx for idx, label in enumerate(self.label_list)}
        train_label_idxs = [label_to_idx[label] for label in train_labels]
        
        # 创建数据集
        train_dataset = IntentDataset(
            train_texts, train_label_idxs, self.tokenizer
        )
        
        train_loader = DataLoader(
            train_dataset, 
            batch_size=batch_size, 
            shuffle=True
        )
        
        # 优化器
        optimizer = torch.optim.AdamW(
            self.model.parameters(), 
            lr=learning_rate
        )
        
        # 损失函数
        criterion = nn.CrossEntropyLoss()
        
        # 训练循环
        self.model.train()
        
        for epoch in range(epochs):
            total_loss = 0
            
            for batch in train_loader:
                input_ids = batch['input_ids']
                attention_mask = batch['attention_mask']
                labels = batch['labels']
                
                # 前向传播
                logits = self.model(input_ids, attention_mask)
                loss = criterion(logits, labels)
                
                # 反向传播
                optimizer.zero_grad()
                loss.backward()
                optimizer.step()
                
                total_loss += loss.item()
            
            avg_loss = total_loss / len(train_loader)
            print(f"Epoch {epoch+1}/{epochs}, Loss: {avg_loss:.4f}")
            
            # 验证（如果有验证集）
            if val_texts and val_labels:
                val_accuracy = self.evaluate(val_texts, val_labels)
                print(f"Validation Accuracy: {val_accuracy:.4f}")
    
    def predict(self, query: str) -> Dict:
        """
        预测意图
        
        Args:
            query: 用户查询
        
        Returns:
            预测结果
        """
        self.model.eval()
        
        # Tokenize
        encoding = self.tokenizer.encode_plus(
            query,
            add_special_tokens=True,
            max_length=128,
            padding='max_length',
            truncation=True,
            return_attention_mask=True,
            return_tensors='pt'
        )
        
        # 预测
        with torch.no_grad():
            logits = self.model(
                encoding['input_ids'],
                encoding['attention_mask']
            )
            
            probabilities = torch.softmax(logits, dim=-1)
        
        # 获取预测结果
        max_prob, predicted_idx = torch.max(probabilities, dim=1)
        
        intent = self.label_list[predicted_idx.item()]
        confidence = max_prob.item()
        
        # 所有类别的概率
        all_probs = {
            label: prob.item()
            for label, prob in zip(self.label_list, probabilities[0])
        }
        
        return {
            'intent': intent,
            'confidence': confidence,
            'all_probabilities': all_probs,
            'method': 'deep_learning_bert'
        }
    
    def evaluate(self, texts: List[str], labels: List[str]) -> float:
        """评估准确率"""
        correct = 0
        
        for text, label in zip(texts, labels):
            result = self.predict(text)
            if result['intent'] == label:
                correct += 1
        
        return correct / len(texts)
    
    def save(self, model_path: str):
        """保存模型"""
        torch.save(self.model.state_dict(), model_path)
        print(f"模型已保存到: {model_path}")
    
    def load(self, model_path: str):
        """加载模型"""
        self.model.load_state_dict(torch.load(model_path))
        print(f"模型已从 {model_path} 加载")
```

```python
import json
from typing import List, Dict

class LLMIntentClassifier:
    """
    基于LLM的意图分类器
    
    支持零样本和少样本学习
    """
    
    def __init__(self, llm_client, model='gpt-4'):
        self.llm = llm_client
        self.model = model
        
        # 意图定义
        self.intent_schema = """
        可用的意图类别：
        - password_reset: 密码重置相关问题
        - order_status: 订单状态查询
        - refund_request: 退款/退货申请
        - product_inquiry: 产品功能咨询
        - technical_support: 技术支持
        - billing_issue: 账单/支付问题
        - account_management: 账户管理
        - general_question: 一般性问题
        - chit_chat: 闲聊
        """
    
    def classify_zero_shot(self, query: str) -> Dict:
        """
        零样本意图分类
        
        无需训练数据，直接使用LLM判断
        """
        prompt = f"""请分析以下用户查询的意图，并从给定的类别中选择最合适的一个。

{self.intent_schema}

用户查询：{query}

请按以下JSON格式输出：
{{
    "intent": "选择的意图类别",
    "confidence": 0.0-1.0之间的置信度,
    "reasoning": "简短的解释为什么选择这个意图"
}}
"""
        
        response = self.llm.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}],
            temperature=0.1,
            max_tokens=200
        )
        
        try:
            result = json.loads(response.choices[0].message.content)
            result['method'] = 'llm_zero_shot'
            return result
        
        except json.JSONDecodeError:
            # 解析失败，返回默认结果
            return {
                'intent': 'general_question',
                'confidence': 0.3,
                'reasoning': '解析失败',
                'method': 'llm_zero_shot'
            }
    
    def classify_few_shot(self, query: str,
                         examples: List[Dict] = None) -> Dict:
        """
        少样本意图分类
        
        Args:
            query: 待分类查询
            examples: 示例列表 [{'query': ..., 'intent': ...}, ...]
        """
        if examples is None:
            examples = [
                {'query': '忘记密码怎么办', 'intent': 'password_reset'},
                {'query': '我的订单到哪了', 'intent': 'order_status'},
                {'query': '我想退款', 'intent': 'refund_request'},
                {'query': '这个产品有什么功能', 'intent': 'product_inquiry'},
            ]
        
        # 构建few-shot prompt
        examples_text = '\n'.join([
            f"查询: {ex['query']}\n意图: {ex['intent']}\n"
            for ex in examples
        ])
        
        prompt = f"""请根据以下示例，判断用户查询的意图。

{self.intent_schema}

示例：
{examples_text}

现在请判断以下查询的意图：
查询: {query}

请按JSON格式输出：
{{
    "intent": "意图类别",
    "confidence": 0.0-1.0,
    "reasoning": "理由"
}}
"""
        
        response = self.llm.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}],
            temperature=0.1
        )
        
        try:
            result = json.loads(response.choices[0].message.content)
            result['method'] = 'llm_few_shot'
            return result
        
        except json.JSONDecodeError:
            return {
                'intent': 'general_question',
                'confidence': 0.3,
                'method': 'llm_few_shot'
            }
```

## 多意图检测

用户查询可能包含多个意图。

```python
class MultiIntentDetector:
    """
    多意图检测器
    
    识别并处理包含多个意图的查询
    """
    
    def __init__(self, single_intent_classifier):
        self.classifier = single_intent_classifier
    
    def detect(self, query: str) -> Dict:
        """
        检测多个意图
        
        Args:
            query: 用户查询
        
        Returns:
            {
                'is_multi_intent': 是否多意图,
                'intents': 意图列表,
                'primary_intent': 主要意图
            }
        """
        # 方法1：分割查询后分别分类
        segments = self._split_compound_query(query)
        
        intents = []
        for segment in segments:
            result = self.classifier.classify(segment)
            
            if result['confidence'] > 0.6:
                intents.append({
                    'segment': segment,
                    'intent': result['intent'],
                    'confidence': result['confidence']
                })
        
        # 方法2：直接让LLM识别多意图（更准确但更慢）
        # intents = self._llm_detect_multi_intent(query)
        
        return {
            'is_multi_intent': len(intents) > 1,
            'intents': intents,
            'primary_intent': intents[0] if intents else None
        }
    
    def _split_compound_query(self, query: str) -> List[str]:
        """分割复合查询"""
        # 基于连接词分割
        connectors = ['并且', '还有', '另外', '以及', '和', ',']
        
        segments = [query]
        
        for connector in connectors:
            new_segments = []
            
            for seg in segments:
                if connector in seg:
                    parts = seg.split(connector)
                    new_segments.extend([p.strip() for p in parts if p.strip()])
                else:
                    new_segments.append(seg)
            
            segments = new_segments
        
        return segments
    
    def _llm_detect_multi_intent(self, query: str) -> List[Dict]:
        """使用LLM检测多意图"""
        prompt = f"""请分析以下查询是否包含多个意图。如果是，请列出所有意图。

查询：{query}

请按JSON格式输出：
{{
    "is_multi_intent": true/false,
    "intents": [
        {{"segment": "子查询1", "intent": "意图1"}},
        {{"segment": "子查询2", "intent": "意图2"}}
    ]
}}
"""
        
        response = self.llm.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}],
            temperature=0.1
        )
        
        result = json.loads(response.choices[0].message.content)
        
        return result.get('intents', [])


# 使用示例
detector = MultiIntentDetector(classifier)

query = "我想重置密码，顺便查一下我的订单状态"

result = detector.detect(query)

print(f"查询: {query}")
print(f"是否多意图: {result['is_multi_intent']}")
print(f"检测到的意图:")
for intent_info in result['intents']:
    print(f"  - [{intent_info['intent']}] {intent_info['segment']}")
```

**处理策略**：
- **串行处理**：依次处理每个意图，合并回复
- **并行处理**：同时处理多个意图，分别返回
- **优先级处理**：只处理主要意图，其他稍后

## 置信度与降级策略

### 多层级意图识别

```python
class IntentHandler:
    """
    意图识别处理器
    
    多层级识别，带降级策略
    """
    
    def __init__(self, rule_classifier, ml_classifier=None,
                 llm_classifier=None):
        self.rule_classifier = rule_classifier
        self.ml_classifier = ml_classifier
        self.llm_classifier = llm_classifier
    
    def classify_with_fallback(self, query: str) -> Dict:
        """
        多层级意图识别，带降级策略
        
        策略：
        1. 规则匹配（快速，高置信度）
        2. ML模型（中等速度，中高置信度）
        3. LLM（慢，高置信度）
        4. Fallback（默认意图）
        """
        
        # 第一层：规则匹配
        rule_result = self.rule_classifier.classify(query)
        
        if rule_result['confidence'] > 0.8:
            return rule_result
        
        # 第二层：ML模型
        if self.ml_classifier and self.ml_classifier.is_trained:
            ml_result = self.ml_classifier.predict(query)
            
            if ml_result['confidence'] > 0.75:
                return ml_result
        
        # 第三层：LLM
        if self.llm_classifier:
            try:
                llm_result = self.llm_classifier.classify_zero_shot(query)
                
                if llm_result['confidence'] > 0.6:
                    return llm_result
            
            except Exception as e:
                logging.warning(f"LLM分类失败: {e}")
        
        # 降级：返回通用意图
        return {
            'intent': 'general_question',
            'confidence': 0.3,
            'method': 'fallback',
            'warning': '低置信度，可能需要人工介入'
        }
```

### 置信度阈值设置

| 置信度范围 | 处理策略 |
|-----------|---------|
| > 0.8 | 直接使用，无需额外处理 |
| 0.6 - 0.8 | 使用，但记录日志供后续分析 |
| 0.4 - 0.6 | 触发澄清问题或提供更多选项 |
| < 0.4 | 降级到通用处理或转人工 |

## 意图驱动的检索策略

根据不同意图采用差异化的检索和处理策略。

```python
class IntentDrivenRetriever:
    """
    意图驱动的检索器
    
    根据意图选择不同的检索配置
    """
    
    def __init__(self, retrievers: Dict):
        self.retrievers = retrievers
    
    def retrieve(self, query: str, intent_info: Dict) -> List[Dict]:
        """
        根据意图执行检索
        
        Args:
            query: 用户查询
            intent_info: 意图识别结果
        
        Returns:
            检索结果
        """
        intent = intent_info['intent']
        
        # 根据意图选择检索策略
        if intent == 'troubleshooting':
            return self._troubleshooting_retrieve(query)
        
        elif intent == 'howto':
            return self._howto_retrieve(query)
        
        elif intent == 'comparison':
            return self._comparison_retrieve(query)
        
        elif intent == 'conceptual':
            return self._conceptual_retrieve(query)
        
        elif intent == 'chit_chat':
            return []  # 闲聊无需检索
        
        else:
            return self._default_retrieve(query)
    
    def _troubleshooting_retrieve(self, query: str) -> List[Dict]:
        """故障排查检索"""
        # 优先匹配错误码和症状
        # 使用关键词增强的检索
        retriever = self.retrievers['keyword_boost']
        
        # 提取错误码
        error_codes = re.findall(r'\b\d{3,4}\b', query)
        
        if error_codes:
            # 有错误码，精确匹配
            results = retriever.search(
                query,
                filters={'error_code': error_codes[0]},
                top_k=10
            )
        else:
            # 无错误码，基于症状匹配
            results = retriever.search(query, top_k=15)
        
        return results
    
    def _howto_retrieve(self, query: str) -> List[Dict]:
        """操作指南检索"""
        # 返回步骤化内容
        retriever = self.retrievers['step_by_step']
        
        results = retriever.search(query, top_k=10)
        
        # 确保结果是步骤化的
        step_results = []
        for result in results:
            if self._is_step_by_step(result['content']):
                step_results.append(result)
        
        return step_results[:10]
    
    def _comparison_retrieve(self, query: str) -> List[Dict]:
        """对比查询检索"""
        # 分解查询，分别检索
        decomposer = QueryDecomposer(llm_client)
        decomposition = decomposer.decompose(query)
        
        all_results = []
        
        for sub_query in decomposition['sub_queries']:
            results = self._default_retrieve(sub_query)
            all_results.extend(results)
        
        # 去重并排序
        seen = set()
        unique_results = []
        
        for result in all_results:
            if result['id'] not in seen:
                seen.add(result['id'])
                unique_results.append(result)
        
        unique_results.sort(key=lambda x: x.get('score', 0), reverse=True)
        
        return unique_results[:20]
    
    def _conceptual_retrieve(self, query: str) -> List[Dict]:
        """概念解释检索"""
        # 使用HyDE增强
        hyde = HyDERewriter(llm_client)
        rewritten = hyde.rewrite(query)
        
        retriever = self.retrievers['vector_dominant']
        results = retriever.search(rewritten['rewritten_query'], top_k=10)
        
        return results
    
    def _default_retrieve(self, query: str) -> List[Dict]:
        """默认检索"""
        retriever = self.retrievers['hybrid']
        return retriever.search(query, top_k=10)
    
    def _is_step_by_step(self, content: str) -> bool:
        """检查内容是否是步骤化的"""
        # 检查是否包含步骤标记
        step_patterns = [r'\d+\.', r'第一步', r'步骤', r'首先', r'然后']
        
        return any(re.search(p, content) for p in step_patterns)
```

### 意图识别评估

```python
class IntentClassificationEvaluator:
    """意图分类评估器"""
    
    def __init__(self):
        pass
    
    def evaluate(self, classifier, test_data: List[Dict]) -> Dict:
        """
        评估意图分类器
        
        Args:
            test_data: [{'query': ..., 'true_intent': ...}, ...]
        """
        metrics = {
            'accuracy': 0,
            'precision_per_class': {},
            'recall_per_class': {},
            'f1_per_class': {},
            'confusion_matrix': {}
        }
        
        # 收集预测结果
        predictions = []
        true_labels = []
        
        for item in test_data:
            query = item['query']
            true_intent = item['true_intent']
            
            result = classifier.classify(query)
            predicted_intent = result['intent']
            
            predictions.append(predicted_intent)
            true_labels.append(true_intent)
        
        # 计算整体准确率
        correct = sum(1 for p, t in zip(predictions, true_labels) if p == t)
        metrics['accuracy'] = correct / len(test_data)
        
        # 计算每类的Precision/Recall/F1
        classes = set(true_labels)
        
        for cls in classes:
            tp = sum(1 for p, t in zip(predictions, true_labels) if p == cls and t == cls)
            fp = sum(1 for p, t in zip(predictions, true_labels) if p == cls and t != cls)
            fn = sum(1 for p, t in zip(predictions, true_labels) if p != cls and t == cls)
            
            precision = tp / (tp + fp) if (tp + fp) > 0 else 0
            recall = tp / (tp + fn) if (tp + fn) > 0 else 0
            f1 = 2 * precision * recall / (precision + recall) if (precision + recall) > 0 else 0
            
            metrics['precision_per_class'][cls] = precision
            metrics['recall_per_class'][cls] = recall
            metrics['f1_per_class'][cls] = f1
        
        return metrics


# 使用示例
evaluator = IntentClassificationEvaluator()

test_data = [
    {'query': '忘记密码', 'true_intent': 'password_reset'},
    {'query': '订单状态', 'true_intent': 'order_status'},
    # ... 更多测试数据
]

metrics = evaluator.evaluate(classifier, test_data)

print(f"准确率: {metrics['accuracy']:.4f}")
print("\n各类别指标:")
for cls in metrics['precision_per_class']:
    print(f"{cls}:")
    print(f"  Precision: {metrics['precision_per_class'][cls]:.4f}")
    print(f"  Recall: {metrics['recall_per_class'][cls]:.4f}")
    print(f"  F1: {metrics['f1_per_class'][cls]:.4f}")
```

## 常见问题

### Q1: 应该选择哪种意图识别方法？

**A**: 
- 初期/小团队：规则方法
- 中期/有数据：ML方法（SVM）
- 成熟期/高精度：深度学习或LLM
- 推荐：多层级组合（规则→ML→LLM）

### Q2: 需要多少训练数据？

**A**:
- 规则方法：0（只需定义规则）
- ML方法：每类50-100个样本
- 深度学习：每类200-500个样本
- LLM：0（零样本）或少量示例（少样本）

### Q3: 如何处理新出现的意图？

**A**:
1. 规则方法：直接添加新规则
2. ML方法：收集数据后重新训练
3. LLM方法：直接在prompt中添加新意图定义

### Q4: 意图识别会增加多少延迟？

**A**:
- 规则方法：< 1ms
- ML方法：< 10ms
- 深度学习：50-100ms（GPU）
- LLM：1000-3000ms

建议使用多层级策略，大部分查询由快速方法处理。

### Q5: 如何评估意图识别的效果？

**A**:
- 离线指标：准确率、Precision、Recall、F1
- 在线指标：用户满意度、任务完成率
- 业务指标：转化率、问题解决率
